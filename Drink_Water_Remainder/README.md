# Drink Water Reminder Project

A Python program that reminds users to drink water at regular intervals.

## Features
- Sends notifications to remind the user to drink water
- Interval and number of reminders can be customized
- Runs in the background using time module

## Technology
- Python
- `time` module
- `plyer` module for notifications
